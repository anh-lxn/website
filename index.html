I’m a mechanical engineering student at TU Dresden and a research assistant in the ITM group for sensor, measurement, and actuator technology. I’m currently exploring data engineering and machine learning to better understand how data can support engineering applications.

Projects

  • Inertial Measurement Unit Teststand

    Development of a two-axis IMU calibration test stand with a differential gear mechanism. Features 3D-printed components, NEMA 17 stepper motors, and precise motion control for sensor calibration.

    Skills used: CAD (Solidworks) · 3D Printing · Embedded Systems (ESP32, DRV8825) · Pandas · Python · Matplotlib · Git

  • Machine Learning Membrane Demonstrator

    Development of a machine learning model for predicting membrane force and position based on data from eight strain sensors. The system runs on a Raspberry Pi 5 and visualizes the predictions in real time on an LED matrix.

    Skills used: Pytorch · Pandas · Matplotlib · Python · Embedded Systems (RaspberryPi 4&5) · Machine Learning

  • Data Processing & Visualization Project

    A small project demonstrating basic data processing and visualization techniques using Python, Pandas and Matplotlib. It includes simple analyses and plots to explore and present dataset structures.

    Skills used: Pandas · Python · Matplotlib
